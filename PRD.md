## 一、产品概述

### 1.1 产品定位

> 输入工作地点，画出通勤圈，一眼看清圈内房价

一个帮助上班族快速找到**通勤友好**住房的决策工具。

### 1.2 解决什么问题

| 场景 | 痛点 | 现有方案 | 我们的方案 |
|------|------|---------|-----------|
| 换工作找房 | 不知道住哪通勤方便 | 逐个小区查通勤时间 | 一圈看清所有选项 |
| 比较多个 offer | 要算每个公司的通勤 | 手动计算 | 同时显示多个圈 |
| 决策效率低 | 信息分散在多个 App | 开 5-6 个应用 | 一个页面搞定 |

### 1.3 目标用户

| 用户类型 | 占比 | 核心需求 |
|---------|------|---------|
| **租房族** | 70% | 快速找到通勤可接受的房子 |
| **买房族** | 20% | 比较多个区域的房价和通勤 |
| **换工作者** | 10% | 评估新工作的通勤成本 |

### 1.4 不做什幺

| 功能 | 原因 |
|------|------|
| ❌ 在线签约 | 太重，有贝壳/链家 |
| ❌ VR 看房 | 技术成本高 |
| ❌ 社交功能 | MVP 不需要 |
| ❌ 复杂筛选 | 第一版简化 |

## 二、功能需求

### 2.1 功能清单（MVP 版）

| 优先级 | 功能 | 说明 | 工时 |
|--------|------|------|------|
| **P0** | 地图展示 | 高德地图，显示北京 | 2h |
| **P0** | 工作地点输入 | 搜索或点击选择 | 2h |
| **P0** | 通勤圈绘制 | 简化版，固定半径 | 3h |
| **P0** | 小区数据展示 | 手动录入 50 个 | 2h |
| **P1** | 价格筛选 | 租金/售价范围 | 2h |
| **P1** | 小区详情 | 点击显示详情 | 2h |
| **P2** | 收藏功能 | 本地存储 | 2h |
| **P2** | 分享功能 | 生成图片 | 3h |

### 2.2 功能详细说明

#### F1: 地图展示（P0）

**用户故事：**
> 作为用户，我想看到北京地图，以便了解整体位置

**功能描述：**
- 加载高德地图
- 默认显示北京（zoom: 11）
- 支持缩放、拖拽

**验收标准：**
- [ ] 地图正常加载
- [ ] 可以缩放和拖拽
- [ ] 加载时间 < 3 秒

---

#### F2: 工作地点输入（P0）

**用户故事：**
> 作为用户，我想输入工作地点，以便计算通勤范围

**功能描述：**
- 搜索框输入地名/小区/写字楼
- 或直接在地图上点击选择
- 显示工作地点标记

**验收标准：**
- [ ] 可以搜索地点
- [ ] 可以点击地图选择
- [ ] 显示工作地点图钉

---

#### F3: 通勤圈绘制（P0）

**用户故事：**
> 作为用户，我想看到通勤范围圈，以便知道哪些地方可以住

**功能描述：**
- 默认 30 分钟通勤圈
- 可选：45 分钟、60 分钟
- 在地图上画圆（简化版）

**验收标准：**
- [ ] 以工作地点为中心画圈
- [ ] 可切换 30/45/60 分钟
- [ ] 圆圈清晰可见

---

#### F4: 小区数据展示（P0）

**用户故事：**
> 作为用户，我想看到圈内小区和价格，以便做决策

**功能描述：**
- 显示圈内所有小区
- 每个小区显示：名称、租金、售价
- 列表 + 地图两种视图

**验收标准：**
- [ ] 圈内小区正确显示
- [ ] 价格信息准确
- [ ] 可切换列表/地图视图

---

#### F5: 价格筛选（P1）

**用户故事：**
> 作为用户，我想按价格筛选，以便找到符合预算的房子

**功能描述：**
- 租金范围滑块
- 售价范围滑块
- 实时筛选结果

**验收标准：**
- [ ] 可以设置价格范围
- [ ] 筛选结果实时更新

---

#### F6: 小区详情（P1）

**用户故事：**
> 作为用户，我想查看小区详情，以便了解更多信​​息

**功能描述：**
- 点击小区标记
- 弹窗显示：名称、地址、租金、售价、通勤时间
- 可收藏

**验收标准：**
- [ ] 点击显示详情
- [ ] 信息完整准确

## 三、技术方案

### 3.1 技术栈

| 模块 | 技术选型 | 说明 |
|------|---------|------|
| **前端框架** | React + TypeScript | 类型安全 |
| **构建工具** | Vite | 快速开发 |
| **地图** | 高德地图 JS API v2.0 | 免费额度够 |

### 3.3 数据模型

```typescript
// 小区数据
interface Community {
  id: string;
  name: string;
  lat: number;
  lng: number;
  rentAvg: number;      // 平均租金（元/月）
  saleAvg: number;      // 平均售价（元/㎡）
  address: string;
}

// 搜索参数
interface SearchParams {
  workLocation: {
    lat: number;
    lng: number;
    name: string;
  };
  commuteTime: 30 | 45 | 60;  // 分钟
  rentRange: [number, number];
}
```
