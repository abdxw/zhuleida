## 🚀 快速部署指南

### 1. 安装 Vercel CLI
```bash
npm install -g vercel
```

### 2. 登录 Vercel
```bash
vercel login
```

### 3. 部署
```bash
cd /app/working/commute-map
vercel --prod
```

### 4. 获取访问链接
部署完成后会显示：
```
🔍  Inspect: https://vercel.com/your-account/commute-map/xxx
✅  Production: https://commute-map.vercel.app
```
